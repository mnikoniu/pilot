# Project Context: Pilot

## Overview

This is the cross-org collaboration pilot project used to validate the three-repository model described in the BMAD implementation proposal.

## Scope

- Validate that external collaborators can set up a multi-root workspace and run BMAD locally
- Confirm shared research, brief, and PRD work happens in git
- Verify source-of-truth boundaries are respected

## Participants

- PCA contributors (access to `docs` nexus + this repo + `pca-bmad-runtime`)
- External collaborators (access to this repo + `pca-bmad-runtime`)
